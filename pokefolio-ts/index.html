Comment jouer ?

Appuyez sur :

  • Z ou ⬆️ pour vous déplacer vers le haut
  • Q ou ⬅️ pour vous déplacer vers la gauche
  • S ou ⬇️ pour vous déplacer vers le bas
  • D ou ➡️ pour vous déplacer vers la droite
  • Shift pour courir
